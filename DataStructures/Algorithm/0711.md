## 1. Checking if a value is Alphanumeric

meaning: consisting of or using both letters and numerals.

```python
 def isAlphanumeric(self,char) -> bool:
        return (ord("A") <= ord(char) <= ord("Z") or
                ord("a") <= ord(char) <= ord("z") or
                ord ("0") <= ord(char) <= ord("9"))
```
