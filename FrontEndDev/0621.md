# Indiana drag scroll

Purpose:
Lightweight Library used to to drag and scroll

Installation:
yarn add react-indiana-drag-scroll

```tsx
import ScrollContainer from "react-indiana-drag-scroll";
<ScrollContainer className="scroll-container flex cursor-pointer">
  <PhilosophyMobile
    videoPath="./video/philosophy1.mp4"
    type={philosophy.mission}
  />
  <PhilosophyMobile
    videoPath="./video/philosophy2.mp4"
    type={philosophy.vision}
  />
  <PhilosophyMobile
    videoPath="./video/philosophy3.mp4"
    type={philosophy.coreValues}
  />
</ScrollContainer>;
```

doc: https://www.npmjs.com/package/react-indiana-drag-scroll

# scroll snap

purpose: used to directly snap to next content at a particular breakpoint.

```tsx
<div className="snap-x flex snap-mandatory overflow-scroll">
  <img
    src="/images/members/cherin.jpg"
    className="flex-shrink-0  snap-center"
  />
```
